Portfolio Hosting over AWS
    Project-03
Technology used: HTML(45%), CSS(45%), Javascript(5%), AWS
Steps:
1. Created a portfolio using HTML, CSS and Javascript.
2. After successful creation of portfolio, Login to Aws console.
3. Search for S3 from the search box.
4. Click to create a bucket.
5. Select Region in my case it is {AsiaPacific(Mumbai)ap-south-1}.
6. Give the unique name of S3 bucket. Or you can choose existing bucket, if you have already.
7. Check object ownership(ACL enables), if it's publicly accessible over internet.
8. Check (uncheck) Block Public access setting to access over internet or manage it according to your need.
9. Enable/Disable bucket versioning to preserve, retrieve, and restore every version of every object stored in your Amazon S3 bucket.
10. Apply tags (optional)
11. Check Encryption default or customize according to your need.
12. (Put it default) or Click on advanced setting and customize according to your need.
13. and lastly click to create bucket.
14. Now, the bucket is successfully created.
15. click on created bucket.
16. click to upload.
17. click to add file or folder.
18. select the file and folder from your device and click to upload.
19. Now, click to close.
20. Now open your folder and click to properties.
21. Scroll down and go to static wbsite hosting section.
22. Enable static website hosting
23. Select Hosting type(static or redirected).
24. scroll down and Give the name of index document
25. Leave error document empty(optional).
26. Click to save changes.
27. Copy the index document link and paste it in browser.
28. Now, It's working.
